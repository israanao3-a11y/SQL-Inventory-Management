# 📊 Gestion de Stock Commercial - SQL

## 📝 Description du projet
Ce projet consiste en la mise en place d'une base de données relationnelle pour la gestion d'un stock de vêtements. Il couvre l'ensemble du cycle, de la création des tables à la génération de rapports de stock dynamiques.

## 🛠 Concepts Techniques Appliqués
* **Modélisation Relationnelle** : Création de schémas (Tables, Primary & Foreign Keys).
* **Requêtes Complexes** : Utilisation intensive de `INNER JOIN` pour lier les produits, catégories et fournisseurs.
* **Logique Métier** : Implémentation de clauses `CASE WHEN` pour créer des alertes de rupture de stock.
* **Vues (Views)** : Création d'un tableau de bord pour visualiser l'état du stock en temps réel.
* **Maintenance des données** : Utilisation de `UPDATE` et `SET` pour automatiser la mise à jour des quantités après réception.

## 🚀 Outils utilisés
* **SGBD** : PostgreSQL / MySQL
* **Outils** : DBeaver, VS Code
* **Version Control** : Git & GitHub
